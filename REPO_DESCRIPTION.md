## üìû Support Strategy

**When people ask questions:**

1. **Installation issues:**
   - Point to: `docs/SETUP_GUIDE.md`
   - Common: Python 3 not installed, permissions
   - Solution: Clear error messages in installer

2. **Discovery failures:**
   - Check: Exa.ai API key (optional but recommended)
   - Check: GitHub token (for private repos)
   - Logs: `~/.mcm/logs/discovery.log`

3. **Feature requests:**
   - Thank them!
   - Label as "enhancement"
   - Phase 1 is complete, Phase 2 is future

4. **Bug reports:**
   - Ask for logs: `~/.mcm/logs/`
   - Ask for MCP that failed
   - Reproduce and fix
